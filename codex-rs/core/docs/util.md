# util.rs

## 文件作用

`util.rs` 提供通用工具函数，包括重试逻辑、错误处理等。

## 主要函数

### backoff()
计算指数退避延迟。

**参数:**
- `retry: usize` - 重试次数

**返回:** `Duration` - 延迟时间

**功能:**
根据重试次数计算递增的延迟时间，用于失败重试。

### error_or_panic()
错误处理或 panic。

**功能:**
根据环境（生产/测试）决定是记录错误还是 panic。

## 其他工具函数

- 字符串处理
- 路径规范化
- 时间戳转换

## 与其他模块的关系

**被依赖模块:**
- 多个模块 - 使用工具函数
- `compact` - 使用 backoff 进行重试
